Пояснення теми: There is / There are в англійській мові

There is
There is
однина (singular) — один предмет або незлічуване
There is a book on the table.
На столі є книга.
There is some water in the glass.
У склянці є вода.
There is a cat in the garden.
У саду є кіт.
There are
There are
множина (plural) — більше одного предмета
There are three books on the table.
На столі є три книги.
There are many students in the class.
У класі багато учнів.
There are two cats in the garden.
У саду є два коти.
Всі форми
Стверджувальна
There is / are …
There is a dog. There are dogs.
Заперечна
There isn't / aren't …
There isn't a cat. There aren't any cats.
Питальна
Is there / Are there …?
Is there a bus? Are there buses?
