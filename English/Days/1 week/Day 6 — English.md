### 📝 Граматичні правила

#### With at

```
Правило: Використовуюэться при вживані здібностей або вмінь
Приклади:
	He`s really good at English
	She`s amazing at the piano
  
```
#### With about

```
Правило: Використовується для того щоб передати почуття людини, як вона зараз себе почуває
Приклади:
	I`m angry about the decision
	He`s nervous about the presentation
```
#### With of

```
Правило: Також використовується для підкреслення почутів
Приклади:
	She was afraid of telling her mum
	She`s scared of flying 
```
#### With to

```
Правило: Для того щоб показати зв'язок між чимось
Приклади:
	He`s married to director
	I`m allergic to nuts
```
#### With for

```
Приклади: 
	The town is fomous for its cheese
	Stress is bad for you
```
#### With in

```
Приклади: 
	She`s interested in the project
	I didn`t want to get involed in the argument
```
#### -Ed adjectives

```
Правило: Описує людину, яка відчуває
Приклади:
	I'm bored
	She's shocked
```

#### -Ing adjectives

```
Правило: Описує річ або ситуацію, що викликає емоцію
Приклади:
	The news is shocking
	The book is boring
```

#### Adjectives with one syllable

```
Правило: To make comparative forms with one-syllable adjectives, we usually add er
Приклади:
	old — older
	clean - cleaner
	slow - slower

Правило: If an adjective ends in -e, we add -r
Приклади:
	safe - safer
	nice - nicer

Правило: If an adjective ends in a vowel and a consonant, we usually double the consonant
Приклади:
	big - bigger
	hot - hotter
```

#### Adjectives with two or more syllables

```
Правило: If a two-syllable adjective ends in a consonant and -y, we change -y to -i and add -er:
Приклади:
	noisy - noisier
	happy - happier
	easy - easier

Правило: We use _more_ to make comparative forms for most other two-syllable adjectives and for all adjectives with three or more syllables
Приклади: 
	crowded - more crowded
	stressful - more stressful
	dangerous - more dangerous

Правило: ou can either add -er/-r or use more with some two-syllable adjectives
Приклади: 
	I think life in the countryside is simpler than in the city.
	It's more simple to live in the city because everything you need is there.
```

#### Irregular adjectives

```
Правило: The adjectives good, bad and far have irregular comparative forms
Приклади:
	good - better
	bad - worse
	far - further
```

#### Than

```
Правило: When we want to say which person or thing we are comparing with, we can use than
Приклади:
	Their house is cleaner than ours.
	Traffic is slower in the city than in the countryside.
	After the race I was more tired than Anne.
```

> [!tip] Додавай нові правила за цим шаблоном нижче
